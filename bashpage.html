Bash Commands

  • "ls" (List) Lists contents of current directory
  • "cd" (Change Directory) Navigates to directory held WITHIN current directory
  • "cd .." Navigates back one level through directory
    • To move up through multiple levels, use "/"
      • For example: "cd ../../../.."
    • Also works moving down
      • For example: "cd documents/bacon_sandwich/Tasty-Bread"
  • "mkdir" (Make Directory) Creates a new directory WITHIN current directory
  • "rm filename" (remove) deletes file
  • "rm -rf directory name (Remove -Recursive_Force) Deletes a directory, even if it contains files
  • "mv" (Move) Moves or Renames a file
    • To move a file enter: "mv filename newlocation"
    • to rename a file enter: "mv filename newfilename"
  • "rm filename" (remove) deletes file
  • "rm -rf directory name (Remove -Recursive_Force) Deletes a directory, even if it contains files
