Variables

Variables store data values

Variable Types:

  • var varName = value:

    Declares a variable


  • let varName = value:

    Variables defined with let cannot be Redeclared. Variables defined with let must be Declared before use. Variables defined with let have Block Scope.

  • const varName = value:

    const variables cannot be changed const variables must be assigned a value when they are declared


    • Arrays

  • var/let/const arrayName = [element1, element2, ... elementN]:

    Arrays are variables that contain multiple elements
    Arrays can hold any type of Primitive Value (string, number, variable,

    • Arrays are index based, starting on 0 ; the first element = 0, the second = 1, the third
    • To specify an index number in an array syntax is: arrayName[indexNumber]
    • Arrays can contain arrays like any other element
    • To specify a nested array the syntax is: arrayName[1stIndex#][2ndIndex#]

    • Objects

  • var/let/const objectName = {
    propertyNameString: "value",
    propertyNameNumber: 1234,
    propertyNameBoolean: True,
    propertyNameArray: [item1, item2, ... itemN]

    • Objects are variables that contain multiple values, written as propertyName:value
    • Properties can store more than one value by using an array
    • To specify a property one can use either:
      1. Bracket notiation: objectName[property]

        2. OR

      2. Dot notation: objectName.property

        3. Objects are NOT index based

        Only property names will work

Loops

Loops run a code block repeatedly until a specific condition is met

Loop Types

  • For Loops: for(expression 1; expression 2; expression 3) { code block to be executed}
    1. is executed (one time) before the execution of the code block.
    2. defines the condition for executing the code block.
    3. is executed (every time) after the code block has been executed.
  • For Of Loops: for(variable of array/object/string { code block to be executed }
    • iterates over selected variable items until reaching the end
  • For In loop
      forvariable in objectName
      • cycles through properties in an object
      forvariable in arrayName
      • cycles through indices on an array
  • While Loops: while(condition) { code block to be executed }
    • Simpler than for loops with only one condition
    • more prone to create an infinite loop, crashing the browser tab
  • Do While Loops do { code block to be executed } while (condition);
    • executes the code block once prior to checking statement
    • then runs until condition is met

Array Methods

method syntax: arrayName.methodName()

  • array.toString(): Converts an array into a comma-seperated string of values: value1,value2,value3;
  • array.join('___'): see toString, but with specified 'delimiter' in parentheses: value1___value2___value3;
  • concatArray = array1.concat(array2, array3): concatenates specified arrays into one new array;
    concatArray = [array1A, array1B, array1C, array2A,array2B, array3A, array3B, etc.]
  • array.splice(index#, #toRemove, 'addItem1', 'addItem2'): removes a selected number fo items from an array an (optional) adds new items in place;
  • newArray = array.slice(index#, #toRemove): cuts items from the specified array and returns them to a newArray
  • Array.indexOf('item/value'): Finds the first index # of an array that contains specified value
  • Array.lastIndexOf('item/value'): Finds the last index # of an array to contain specified value
  • Array.forEach(): Loops through the given array, and calls the given call-back function for each of the elements in the array.
  • Array.map(): The map function takes in a call-back function as argument, and executes that function on each element of the array it is working on. It maps each of the return values of the call-back into a new array. It does not mutate the original array.
  • Array.filter():The filter method is used to filter out array elements that fail a boolean test. Only elements that pass the test are allowed through into the new return array.
