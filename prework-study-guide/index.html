HTML

  • The HEAD element contains information about the webpage
  • Information in the HEAD is not visible to the end user
  • Information within HEAD is for the web browser to read and display the webpage properly
  • The BODY element represents the visible content to the end user
  • HTML acts as the foundation of webpages

Git

  • ALL inputs relating to Git or Github must begin with "git"
  • "git log" shows the commit history of the currently active branch
  • "git status" shows modified files in working directory, staged for the next commit
  • "git branch" lists branches with an * appearing on the active branch
  • "git checkout" switches to another branch and checks the branch out into working directory
  • "git checkout -b new-branch-name-example" creates a new branch and switches to it
