Array Methods

HOME BACK

method syntax: arrayName.methodName()

  • array.toString(): Converts an array into a comma-seperated string of values: value1,value2,value3;
  • array.join('___'): see toString, but with specified 'delimiter' in parentheses: value1___value2___value3;
  • concatArray = array1.concat(array2, array3): concatenates specified arrays into one new array;
    concatArray = [array1A, array1B, array1C, array2A,array2B, array3A, array3B, etc.]
  • array.splice(index#, #toRemove, 'addItem1', 'addItem2'): removes a selected number fo items from an array an (optional) adds new items in place;
  • newArray = array.slice(index#, #toRemove): cuts items from the specified array and returns them to a newArray
  • Array.indexOf('item/value'): Finds the first index # of an array that contains specified value
  • Array.lastIndexOf('item/value'): Finds the last index # of an array to contain specified value
  • Array.forEach(): Loops through the given array, and calls the given call-back function for each of the elements in the array.
  • Array.map(): The map function takes in a call-back function as argument, and executes that function on each element of the array it is working on. It maps each of the return values of the call-back into a new array. It does not mutate the original array.
  • Array.filter():The filter method is used to filter out array elements that fail a boolean test. Only elements that pass the test are allowed through into the new return array.
